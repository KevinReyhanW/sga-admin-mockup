# Super Admin SGA - Mockup

Premium management dashboard for hotel administration, built with Vite, React, and Tailwind CSS v4.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite 6 (Downgraded for Tailwind v4 compatibility)
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts
- **Animations**: Framer Motion
- **State**: Zustand

## Deployment
This project is optimized for Vercel deployment. 

> [!NOTE]
> Vite has been anchored to version 6 to ensure compatibility with `@tailwindcss/vite` until Vite 8 support is finalized.
